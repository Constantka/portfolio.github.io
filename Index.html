School Web

Project Overview:

The School Web Project (May 2020 - December 2021) is a digital platform designed to address challenges in education within the Democratic Republic of the Congo, particularly during the COVID-19 pandemic. Kidney Stone Detection

Project Overview:

The Kidney Stone Detection Project (2023) is a machine learning initiative to enhance kidney stone detection in X-ray images using an improved YOLOv7 architecture with CBAM attention and super-resolution methods like EDSR and ESRGAN. It addresses challenges such as low X-ray quality and small object detection, achieving up to 97.3% precision and 95.0% mAP50, improving diagnostic accuracy in resource-constrained settings like the DRC.

My Role:

As the sole Developer and Researcher, I designed and implemented the YOLOv7-based model with CBAM integration, developed super-resolution techniques to enhance X-ray image quality, and conducted experiments to evaluate model performance, ensuring robust kidney stone detection for healthcare applications.

Kin Virtual Tour

Project Overview:

The Kin Virtual Tour Project (2022-2023) is a digital platform designed to provide immersive virtual tours of Kinshasa, Democratic Republic of the Congo, showcasing its cultural, historical, and educational landmarks. Leveraging AI and data science, the platform uses image recognition to identify and label landmarks, NLP for multilingual audio narration, and data analytics to enhance user engagement, making Kinshasa accessible to a global audience while addressing connectivity challenges in the region.

My Role:

As the Lead Project Manager and AI Developer, I oversaw the Kin Virtual Tour Project, coordinating the development team, designing the AI-driven features like image recognition and NLP for multilingual support, and implementing data analytics to optimize user experience, ensuring the platform effectively promotes Kinshasa’s heritage.
